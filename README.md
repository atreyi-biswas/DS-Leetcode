# DS-Leetcode

A curated collection of essential LeetCode problems for Data Science and Analytics interviews, covering Python, SQL, and fundamental data structures and algorithms commonly asked in Data Analyst and Data Scientist roles.
**All DSA problems are done using C++**

## Topics Covered

* Arrays
* Strings
* Hash Tables
* Sorting
* Searching
* Stack
* Queue
* Linked Lists
* Trees
* Recursion
* Heaps
* Matrices
* Intervals
* SQL


## Recommended Preparation Order

```text

Strings

↓

↓
```
